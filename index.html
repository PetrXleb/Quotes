MY WORKS

Timetable

One of my most time-consuming projects. The idea to create a schedule application appeared by itself, since in our university the schedule is divided into two groups and two weeks. It is quite difficult to navigate by a regular pdf file at first. In addition to displaying classes, there is also a time before these classes, for better orientation. The alternation of weeks occurs automatically, but if desired, you can do it yourself. There are several types of color design, you can also conveniently scroll through the phone using a scroll, and there are many more small chips that I don't have enough space to paint.

Расписание

Один из моих самых временно затратных проектов. Идея создать приложение расписание появилась сама собой, так как в нашем вузе расписание разбито на две группы и две недели. Довольно сложно ориентироваться по обычному pdf файлу в первое время. Помимо отображения занятий, предусмотрено и время до этих занятий, для лучшей ориентации. Чередование недель происходит автоматически, но при желании это можно осуществить самому. Есть несколько видов цветового оформления, также на телефоне можно удобно листать с помощью скролла, и еще много мелких фишек, которых мне не хватит места расписать.

Dictionaty With Translator

This project in theory can help beginners in learning English by memorizing flashcards with words. In this project, I learned how to work with an API that allows you to translate text from one language to another. I also tried regular expressions for the first time. Using the API, I made something like an English-Russian translator. The words from the cards are translated and recorded in the list below, for further memorization

Словарь/Переводчик

Этот проект в теории может помочь новичкам в изучении английского языка, путём запоминания карточек со словами. В этом проекте я научился работать с API интерфейсом, позволяющим переводить текст с одного языка на другой. Также впервые попробовал регулярные выражения. С помощью API я сделал что-то вроде англо-русского переводчика. Слова из карточек переводятся и записываются в список ниже, для дальнейшего их запоминания

MusicTest

The first job done with React. The application consists of a test of 25 questions and a leaderboard at the end. The player database is stored on the MockAPI service. The main difficulty was retraining, it was difficult for me to do simple things that would have been done in a matter of minutes on pure JS. But nevertheless, the project gave the first experience in the development of React applications. I deployed work on the Netlify platform, and you can see the code here https://github.com/PetrXleb/MusicTest

Музыкальный тест

Первая работа, выполненная с помощью React. Приложение представляет из себя тест из 25 вопросов и таблицей лидеров в конце. База игроков хранится на сервисе MockAPI. Главная сложность была в переобучении, мне было сложно делать простые вещи, которые на чистом JS были бы сделаны за считанные минуты. Но тем не менее, проект дал первый опыт в разработке React приложений. Развернул работу на платформе Netlify, а код можете посмортеть тут https://github.com/PetrXleb/MusicTest

Visual Novel

This project is an engine for writing browser-based visual novels. The idea to create something like this came from my love of visual novels. You should not count on any design, the emphasis was on functionality. According to the idea, anyone who doesn't even know much coding can write a novel using my ready-made functions to change the background, music, character name, and so on. I also provided a simple save. There is still a lot of functionality to be added, but now you can try to create a small novel

Визуальная новелла

Этот проект представляет из себя движок для написания браузерных визуальных новелл. Идея создать что-то подобное появилась из моей любви к визуальным новеллам. На какой-либо дизайн рассчитывать не стоит, упор делался на функционал. По идеи любой человек, даже не владеющий особо кодингом, может написать новеллу, используя мои готовые функции для изменения фона, музыки, имени персонажа и так далее. Также я предусмотрел простенькое сохранение. Еще много функционала предстоит добавить, но уже сейчас можно попробовать создать небольшую новеллу

EasyQuotes

The idea to create this site appeared by chance. My friends and I like to come up with various local "quotes", but they were quickly forgotten if they were not written down anywhere. Then I decided to make something like a web notebook. In this project, I learned how to use the mocAPI service, which allows you to make test requests to the server using fetch in my case. Quotes can be added, existing ones can be changed, deleted (the deletion password is "999"). When adding a new one, it must be saved

Цитатник

Идея создать этот сайт появилась случайно. Мои друзья и я любим придумывать различные локальные "цитаты", но они быстро забывались, если их никуда не записывать. Тогда я решил сделать что-то вроде web блокнота. В этом проекте я научился пользоваться сервисом mocAPI, который позволяет делать тестовые запросы на сервер, используя в моем случаи fetch. Цитаты можно добавлять, изменять существующие, удалять (пароль удаления "999"). При добавлении новой ее нужно обязательно сохранить
